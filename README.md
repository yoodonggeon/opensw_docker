# opensw_docker

Django와 Docker를 사용하여 6/45 Lotto 웹 사이트를 개발합니다.

- 일반 사용자는 복권 구매(수동 번호 및 자동 번호), 당첨 확인 기능 제공

- 관리자는 판매 내역 확인, 추첨 기능, 당첨 내역 확인 기능 제공

- 서비스 환경은 Docker를 사용하여 multi-container로 구성할 것

* 시스템 설계, 구현 과정, 테스트 결과를 상세히 기록한 보고서를 제출할 것

* 소스 코드는 보고서 내에 GitHub 링크로 명시할 것

* AI 도구를 사용할 경우 사용 내역을 명확히 제시할 것
